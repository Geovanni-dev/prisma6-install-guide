⚡ Prisma 6 — Configuração Completa

Comandos para configurar o Prisma 6 com SQLite ou PostgreSQL

Prisma 6 | SQLite | PostgreSQL
SQLite 📦 Instalação e Configuração
#ComandoDescrição
1npm install prisma@6 --save-devInstala o Prisma CLI como dependência de desenvolvimento
2npm install @prisma/client@6Instala o Prisma Client para sua aplicação
3npx prisma init --datasource-provider sqliteInicializa o Prisma com SQLite (cria pasta prisma/ e .env)
4npx prisma migrate dev --name initCria a primeira migração e gera o cliente
5 npx prisma generate --schema=prisma/schema.prisma ⚠️ FORÇA O PRISMA 6 — Gera o cliente forçando o schema do Prisma 6

🔧 Comandos do Dia a Dia (Ambos Bancos)
ComandoQuando usar
npx prisma migrate dev --name nomeApós alterar o schema (cria nova migração)
npx prisma generateApós instalar em outro ambiente ou atualizar o cliente
npx prisma db pushPara desenvolvimento rápido (sem criar migrações)
npx prisma migrate resetReseta o banco e aplica todas as migrações
npx prisma studioAbre interface visual do banco de dados
npx prisma validateVerifica se o schema está válido
npx prisma formatFormata o arquivo schema.prisma

📁 Estrutura de Pastas
Arquivo/PastaDescrição
prisma/schema.prismaSeu schema com os models
prisma/migrations/Histórico de migrações (criado automaticamente)
prisma/dev.dbArquivo do banco SQLite (criado automaticamente)
.envVariáveis de ambiente com a DATABASE_URL
SQLite 📝 Exemplo de schema.prisma 
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "sqlite"
  url      = env("DATABASE_URL")
}

model User {
  id    Int    @id @default(autoincrement())
  email String @unique
  name  String
}
PostgreSQL 📝 Exemplo de schema.prisma 
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}

model User {
  id    Int    @id @default(autoincrement())
  email String @unique
  name  String
}
🔐 Exemplo de arquivo .env
# SQLite - Banco local, sem instalação
DATABASE_URL="file:./prisma/dev.db"

# PostgreSQL - Banco robusto pra produção
DATABASE_URL="postgresql://usuario:senha@localhost:5432/nome_do_banco"
