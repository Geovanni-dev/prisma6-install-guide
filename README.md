## 📋 Sobre

Este repositório contém um guia visual e prático para configurar o **Prisma 6** do zero, com suporte a **SQLite** (banco local) e **PostgreSQL** (banco robusto para produção). O material é apresentado em formato de tabela dark theme, ideal para consulta rápida durante o desenvolvimento.

**Por que Prisma 6?**  
Estável, maduro, documentação consolidada e sem as quebras de compatibilidade do Prisma 7.
